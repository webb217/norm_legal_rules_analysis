# Norm Legal Rules Analysis

## Overview

This repository contains supporting materials for the legal rules analysis exercise.

The analysis focused on the evaluation of cross-jurisdictional consistency and ambiguity, risk exposure within legal rules, and the identification of opportunities to improve the data's structural integrity and downstream usability.
